# Eager Wrapper Gotcha.io Implementation

This is a Gotcha.io app for [Eager](https://eager.io).

This app inserts an embed code from [Gotcha](https://www.gotcha.io), user-centric analytics & triggers, driven by micro-surveys
SaaS provider.  To create a free account head over to [www.gotcha.io](https://www.gotcha.io).
